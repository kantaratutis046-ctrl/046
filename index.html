เลือกชุมชนที่รับผิดชอบ
แต่ละเครื่องควรเลือกชุมชนของตัวเอง
เพื่อป้องกันข้อมูลทับกันตอน Sync
💡 วิธีใช้งาน: ตั้งชื่อทีม → เลือกชุมชน → กด Sync เฉพาะชุมชนของตัวเอง
🏠 กดปุ่ม "📌 รับบ้านนี้" ในแต่ละบ้านเพื่อจองว่าทีมไหนดูแล ป้องกันทีมซ้ำกัน
📍 เลือกชุมชนที่ดูแล (เลือกได้หลายชุมชน)
⚙️ จัดการข้อมูล
📤 ส่งออก / นำเข้า
🔗 Google Sheets
📖 วิธีตั้งค่า Apps Script →
⚠️ อันตราย
📊 สรุปภาพรวมทุกชุมชน
📖 วิธีตั้งค่า Google Sheets
  1. เปิด Google Sheets ใหม่ → Extensions → Apps Script
  2. ลบโค้ดเดิมออก วางโค้ดด้านล่างแทน แล้วกด Save 💾
  3. กด Deploy → New deployment → ประเภท: Web app
  4. Execute as: Me · Who has access: Anyone
  5. กด Deploy → คัดลอก URL มาวางในช่อง Script URL ด้านบน
  6. ✅ Sync แล้ว Google Sheets จะสร้างแถบแยกอัตโนมัติ 9 ชุมชน + สรุปรวม
